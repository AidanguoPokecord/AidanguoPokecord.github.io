Notes 4 - Semantic elements

  • text-align(left, right, center), specifies horizontal alignment of text,

  • text-decoration(none, undeline, overline, line through), decorate your text.

  • font-weight(normal 400, bold 700, 100-900), make font bolder or lighter.

  • font-family, specifies the font for an element.

  • span, for grouping and inline elements

  • id, specifies unique id for element. can be used on any html element.

  • #, to start id off with
